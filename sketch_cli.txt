Usage: dotf-go <command> <args> [--help]

Commands:
	move		to from		inits the dotfiles dir at a new location
	add			file/dir	adds new file/dir to the dotiles repo
	install		file/dir	installs a dotfile symlink into the system
	restore		symlink		restores file to system by moving dotfile back

Examples:
	dotf-go move --help			shows help for the move command

#INDIVIDUAL COMMANDS:
-------------------
Name
	dotf-go move - update userspace symlinks to point to another dotfiles location

Usage: 
	dotf-go move [<options>...] <dotfiles-dir-path> <userspace-dir-path> [--help]

Description:
	Iterates through all files in 'from' and updates matching symlinks in 'to'.

Arguments:
	<dotfiles-dir-path>		Path specifies re-located dotfiles directory.
	<userspace-dir-path>		Path specifies userspace directory where symlinks will be updated.

Options:
	-s, --shared
		Apply something to the shared dotfiles
